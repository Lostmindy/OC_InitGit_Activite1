==================================
== Initiation à Git
=========== Annexes
==================================

1 - Utilisation de l'invite de commandes
Commandes 
	• cat  	// pour lire un fichier
	• touch	// pour créer un fichier
	
Copier-coller le sha
	L'invite de commande n'aime pas le copier coller. Il faut faire un clic droit dans l'en-tête de la fenêtre puis faire modifier » sélectionner, sélectionner le texte souhaité, et enfin coller en allant cliquer sur modifier » coller.

2 - Utilisation de vim

En faisant ECHAP, on entre en mode interactif
	:w permet d'écrire (enregistrer) le fichier
	:q permet de quitter vim

En tapant i, on entre en mode écriture
